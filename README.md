# 3D Interactive Developer Portfolio

An immersive, interactive 3D web-based developer portfolio designed to showcase skills, projects, and a deep understanding of frontend development, 3D graphics, and user experience.

## 🔥 Features
- **🧭 Mouse-based 3D Movement**: Elements tilt, shift, and rotate based on mouse interaction, creating a true depth illusion.
- **🧱 Floating 3D Cards**: Glassmorphism UI components that lift and change shadow intensity when hovered.
- **🌌 Parallax 3D Background**: A Three.js particle system and geometric core that react to scroll and mouse movements.
- **⚡ Smooth Animations**: Powered by GSAP and ScrollTrigger for seamless section transitions.
- **🌓 Dark/Light Theme**: Fully responsive color palette toggle.

## 💻 Tech Stack
- **HTML5 & CSS3** (Vanilla CSS with Custom Properties and Glassmorphism)
- **JavaScript (ES6+)**
- **Three.js** (for 3D scene, cameras, geometries, and materials)
- **GSAP** (for complex sequencing and scroll-based animations)

## 🛠 Setup Instructions
Since this project uses pure HTML/CSS/JS with CDNs, there is no complex build step required.

1. Clone the repository:
   ```bash
   git clone https://github.com/annanyabagga01/3DInteractivePortfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd 3DInteractivePortfolio
   ```
3. Open `index.html` in your browser, or use an extension like **Live Server** in VS Code.
